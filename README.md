### [일러두기 1]
이 책의 실습을 시작하기 전에 다음에 주의해주세요.
1. 클로드 코드는 작업할 때마다 권한을 묻습니다. ‘Yes/No?’, ‘Do you want to proceed?’ 같은 질문이 자
주 뜨는 게 원래 정상입니다. 당황하지 마세요.
2. 이 책은 모든 권한 질문에 2번(Yes, and don’t ask again)을 선택해 진행합니다. 한 번 허용하면 같은 폴
더에서는 다시 묻지 않아서 진행이 빨라집니다. 1번을 선택해도 결과는 같습니다.
3. 자료실에서 내려받은 CLAUDE.md 파일을 실습 폴더에 두면 질문 횟수가 80% 이상 줄어듭니다.
4. 결과가 본문 화면과 100% 같지 않습니다. AI는 매번 조금씩 다른 결과를 만들기 때문에, 핵심 산출물(
travel_plan.md 생성 여부)만 확인하면 됩니다.
5. 클로드 코드 작업이 너무 길어지거나 도중에 헤매는 느낌이면 Esc 로 중단하고, 같은 프롬프트를 다시 입
력하세요. 처음보다 빨리 끝날 때가 많습니다.
