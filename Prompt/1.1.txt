가족 여행 플래너를 만들 거야.
사용자가 입력하는 정보는 이렇다:
- 여행지 (예: 제주도, 부산, 강릉)
- 여행 기간 (예: 2박 3일, 3박 4일)
- 총 예산 (예: 80만원, 150만원)
- 가족 구성 (예: 성인 2명, 아이 2명(5세, 8세))
- 여행 스타일 (예: 자연 중심, 맛집 투어, 테마파크)
이 정보를 받아서 travel_input.json 파일로 저장하는 파이썬 스크립트를 만들어 줘.



오픈클로 에이전트를 만들어줘. 이름은 travel-course-agent.
이 에이전트는 travel_input.json을 읽어서 일자별 여행 코스를 생성해야 해.
각 일자마다 아래 내용이 포함되어야 해:
- 오전/오후/저녁 관광지 추천 (아이 동반 가능 여부 표시)
- 관광지 간 이동 시간 (대략적)
- 추천 식당 (가족 친화적인 곳)
- 일자별 예상 비용 (입장료 + 식비 + 교통비)
웹 검색 도구를 사용해서 실제 관광지 정보를 검색하고,
결과를 travel_course.json으로 저장해줘.



숙소 추천 에이전트를 추가해줘. 이름은 accommodation-agent.
travel_input.json에서 예산과 가족 구성을 읽고,
travel_course.json에서 여행 코스를 참고해서
아래 기준으로 숙소를 추천해줘:
- 총 예산의 40%를 숙박비로 배정
- 가족 구성에 맞는 객실 타입 (예: 패밀리룸, 키즈 프렌들리)
- 당일 관광지와 가까운 위치 우선
- 가격대별 3단계 옵션 (절약형/표준형/프리미엄)
웹 검색으로 실제 숙소 정보를 찾고,
결과를 accommodation.json으로 저장해 줘.



예산 검증 기능을 추가해 줘.
travel_course.json의 일자별 비용 합계 + accommodation.json의 숙박비 총합이 travel_input.json의 총 예
산을 초과하면 자동으로 조정안을 제시해 줘.
조정 방법:
1순위 - 숙소 등급 하향
2순위 - 유료 관광지를 무료 대안으로 교체
3순위 - 외식 횟수 줄이기 (편의점/도시락 대안)
최종 예산 요약을 budget_summary.json으로 저장해 줘.



travel_course.json, accommodation.json, budget_summary.json을 종합해서 예쁜 여행 계획서를 만들어 줘.
포함할 내용:
- 제목: "우리 가족 제주도 여행 계획서"
- 일자별 상세 일정표 (시간, 장소, 이동 수단)
- 숙소 정보 (이름, 주소, 체크인/체크아웃 시간)
- 예산 요약표 (항목별 비용 + 총합)
- 준비물 체크리스트 (아이 연령대에 맞게)
마크다운 파일 travel_plan.md로 만들어줘.



지금까지 만든 것들을 하나의 오픈클로 워크플로로 묶어 줘.
실행 순서:
1. 사용자 입력 받기 -> travel_input.json 생성
2. travel-course-agent 실행 -> travel_course.json 생성
3. accommodation-agent 실행 -> accommodation.json 생성
4. 예산 검증 -> budget_summary.json 생성
5. 최종 계획서 생성 -> travel_plan.md 생성
워크플로 이름은 family-travel-workflow로 해 줘.



family-travel-workflow를 실행해 줘.
