# CLAUDE.md 팩 — 실습용 자동 동작 지침서

이 폴더는 책 *실습편*의 1.1 ~ 3.5 (총 10개 실습)에서 클로드 코드가 **확인 질문을 최소화하고 끝까지 자동으로 진행**하도록 만든 보조 파일 묶음입니다. 출판사 측 검토 결과 클로드 코드가 너무 자주 질문하는 문제(독자 1.1 실습에서 30회 이상)를 줄이기 위한 패키지입니다.

## 2. 독자 배포 시 권장 구조

각 실습 폴더를 만든 직후, 그 안에 두 파일을 넣어 두기만 하면 됩니다.

```
D:\claudecode\family-travel-planner\
├── CLAUDE.md                ← 실습 1.1용 (이 팩의 1.1-family-travel-planner/CLAUDE.md)
└── .claude\
    └── settings.json        ← 공통 settings.json (이름 동일)
```

> `.claude` 폴더가 보이지 않으면 윈도우 탐색기에서 "숨김 파일 표시"를 켭니다. 명령 프롬프트에서는 `mkdir .claude` 로 만들 수 있습니다.

## 3. 무엇이 달라지나

### (1) CLAUDE.md — 매 세션 자동으로 읽히는 작업 지침
각 실습 폴더의 CLAUDE.md는 다음 7개 섹션을 갖습니다.

1. **가장 중요한 작업 원칙** — "확인 질문 금지, 기본값으로 끝까지 진행"
2. **프로젝트 목표** — 한 줄 요약 + 최종 워크플로우 이름
3. **산출물 명세** — 실습 종료 시점에 존재해야 하는 파일 목록
4. **사전 결정된 기본값** — 언어·인코딩·통화·주기 등 모든 작은 결정을 미리 박아둠
5. **절대 하지 말 것** — 임의 패키지 설치, 폴더 외부 접근, 책에 없는 기능 추가 금지
6. **작업 흐름 매핑** — 책에 나오는 사용자 프롬프트 키워드 → 클로드가 즉시 해야 할 행동
7. **실패·모호함 처리** — 막혔을 때 사용자에게 묻지 않고 자동 결정하는 규칙
8. **완료 보고 형식** — 군더더기 없는 4~5줄 요약

### (2) settings.json — 권한 사전 허용
- 파일 읽기·쓰기·편집, mkdir, python 실행 등 책 실습에서 반드시 쓰는 도구를 미리 허용
- 위험 명령(`rm -rf`, `format`, `shutdown` 등)은 명시적으로 차단
- 외부 발송(웹 fetch, MCP 통한 메일·캘린더 발송)은 의도적으로 빠져 있어 처음 1회만 묻습니다 (보안)

## 4. 출판사 권장 수록 방식

세 가지 옵션 중 택일을 권장합니다.

| 옵션 | 방식 | 장점 | 단점 |
|------|------|------|------|
| A | 책 부록 + 길벗 자료실 다운로드 ZIP | 가장 깔끔, 업데이트 용이 | 독자가 다운로드 절차 추가로 거쳐야 함 |
| B | 각 실습 시작 페이지에 "CLAUDE.md 전체 코드" QR 코드 + 자료실 링크 | 인쇄 부담 적음 | 독자가 QR 스캔 필요 |
| C | 각 실습 도입부에 CLAUDE.md 본문을 박스로 인쇄 + 자료실 링크 병기 | 책만 봐도 됨 | 페이지 수 증가 (실습당 1~2페이지) |

**추천: A (자료실 ZIP) + 본문에 "실습 시작 전 CLAUDE.md를 폴더에 복사하세요" 안내 박스 1단락**

## 5. 본문에 함께 안내하면 좋은 시작 프롬프트

CLAUDE.md만 두어도 효과가 크지만, 시작 프롬프트도 통일된 양식으로 제공하면 헤매는 폭이 더 줄어듭니다. 각 실습 첫 프롬프트 직전에 다음 안내를 1회만 넣으시면 됩니다.

```
[실습 시작 전 안내]
이 폴더에는 CLAUDE.md가 있습니다.
클로드 코드는 자동으로 이 지침을 읽고
확인 질문 없이 끝까지 진행하도록 동작합니다.
중간에 'yes/no' 질문이 보이면 모두 2번(Yes) 또는 Enter를 누르세요.
```

## 6. 효과 추정

- 1.1 실습 기준: **확인 질문 30회 → 3~5회 수준** (외부 서비스 호출 시점만 남음)
- 1.2 실습 기준: MCP 인증 1회 + 캘린더 등록 1회 = **최대 2~3회**
- 토큰 사용량: 불필요한 탐색·재확인 감소로 **40~60% 절감 추정**
- 진행 시간: 1.1 기준 2시간 → **20~30분** 으로 단축 가능 (외부 API 응답 시간 제외)

위 수치는 책의 실습 내용을 기준으로 한 합리적 추정치이며, 독자 환경(네트워크·MCP 인증 상태)에 따라 달라집니다.

## 7. 자주 묻는 질문

**Q. CLAUDE.md 없이도 책 실습이 동작하나요?**
A. 네, 동작합니다. CLAUDE.md는 "물어보는 횟수를 줄여주는 보조 가이드"이지 필수 파일이 아닙니다. 본문 실습은 CLAUDE.md 없이도 그대로 진행 가능합니다.

**Q. CLAUDE.md를 수정해도 되나요?**
A. 됩니다. 특히 "사전 결정된 기본값" 섹션은 독자의 환경(예: 통화, 언어)에 맞게 수정하면 됩니다.

**Q. settings.json의 `deny` 목록을 풀어도 되나요?**
A. 권장하지 않습니다. 책 실습에서 그 명령들은 필요하지 않으며, 실수로 시스템 폴더를 지우는 사고를 막아 줍니다.

**Q. 클로드가 그래도 자꾸 묻는다면?**
A. 사용자 프롬프트 첫 줄에 `"이 폴더의 CLAUDE.md를 먼저 읽고 그 지침을 따라줘. 추가 확인 없이 끝까지 진행해줘."`를 1회 덧붙여 보세요. 이후 세션에서는 자동으로 적용됩니다.
